# 🔍 Named Entity Recognition (NER) App

A web app that detects named entities (Person, Organization, Location, Misc) from text using Hugging Face Transformers and Gradio.

## 🚀 Live Demo
👉 [Click here to try the app](https://huggingface.co/spaces/Obhagat01/NER_APP)

## Features
- Detects People, Organizations, Locations, Miscellaneous entities
- Color-coded highlighting
- Confidence scores for each entity
- Built with BERT + Gradio

## How to Run Locally
pip install -r requirements.txt
python app.py

## Model Used
`dbmdz/bert-large-cased-finetuned-conll03-english`
